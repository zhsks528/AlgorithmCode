## 백준알고리즘 - 2754. 학점계산

- 관련된 알고리즘 : .

### 문제

어떤 사람의 C언어 성적이 주어졌을 때, 평점은 몇 점인지 출력하는 프로그램을 작성하시오.

A+: 4.3, A0: 4.0, A-: 3.7

B+: 3.3, B0: 3.0, B-: 2.7

C+: 2.3, C0: 2.0, C-: 1.7

D+: 1.3, D0: 1.0, D-: 0.7

### 입력

첫째 줄에 C언어 성적이 주어진다. 성적은 문제에서 설명한 13가지 중 하나이다.

### 출력

첫째 줄에 C언어 평점을 출력한다.

### 예제

```
input : A0
output : 4.0
```

### 해결

```python
n = input()

a = {'A+': 4.3, 'A0': 4.0, 'A-': 3.7,
     'B+': 3.3, 'B0': 3.0, 'B-': 2.7,
     'C+': 2.3, 'C0': 2.0, 'C-': 1.7,
     'D+': 1.3, 'D0': 1.0, 'D-': 0.7,
     'F': 0.0}

print(a[n])

```

### 설명

1. dict을 이용하여 학점에 대한 평점을 저장한다.
