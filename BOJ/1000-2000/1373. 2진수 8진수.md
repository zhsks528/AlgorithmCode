## 백준알고리즘 - 1373. 2진수 8진수

- 관련된 알고리즘 : .

### 문제

2진수가 주어졌을 때, 8진수로 변환하는 프로그램을 작성하시오.

### 입력

첫째 줄에 2진수가 주어진다. 주어지는 수의 길이는 1,000,000을 넘지 않는다.

### 출력

첫째 줄에 주어진 수를 8진수로 변환하여 출력한다.

### 예제

```
Input : 11001100
Output : 314
```

### 해결

```python
n = input()

b = int('0b' + n, 2)  # 2진수 -> 10진수 변환

res = format(b, 'o')  # 10진수 -> 8진수 변환

print(res)
```
